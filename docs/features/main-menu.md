# 메인 메뉴 SPEC

## 개요

- 기능별 선택 화면을 Display
- 전체 시료에 대한 요약 정보를 확인

---

## 메뉴 구성

| 번호 | 메뉴명 | 설명 |
|------|--------|------|
| 1 | 시료 관리 | 새로운 시료 등록, 목록 조회, 이름 검색 기능 |
| 2 | 시료 주문 | 고객 주문 접수 |
| 3 | 주문 승인/거절 | 생산 라인 담당자의 승인·거절 처리 |
| 4 | 모니터링 | 상태별 주문 수 및 시료별 재고 현황 확인 |
| 5 | 생산라인 조회 | 현재 생산 중인 시료 및 대기 중인 생산 큐 확인 |
| 6 | 출고 처리 | CONFIRMED 상태 주문에 대해 출고 실행 |
| 0 | 종료 | 시스템 종료 |

---

## 시스템 현황 요약

메인 메뉴 진입 시 상단에 아래 항목을 표시한다.

| 항목 | 설명 |
|------|------|
| 현재 일시 | 시스템 현재 시각 |
| 등록 시료 | 등록된 시료 총 종류 수 |
| 총 재고 | 전체 시료 재고 합계 (ea) |
| 전체 주문 | 등록된 전체 주문 건수 |
| 생산라인 대기 | 현재 생산 큐에 대기 중인 건수 |

---

## UI 화면 예시

```
============================================================
          반도체 시료 생산주문관리 시스템
============================================================
 시스템 현황   2026-04-16 09:32:15

  등록 시료   12종      총 재고      2,840 ea
  전체 주문   36건      생산라인     3건 대기
------------------------------------------------------------
  [1] 시료 관리              [2] 시료 주문
  [3] 주문 승인/거절          [4] 모니터링
  [5] 생산라인 조회           [6] 출고 처리
  [0] 종료
------------------------------------------------------------
  선택 > _
```

> 위 화면 구성은 예시이며 자유롭게 표기 가능.
