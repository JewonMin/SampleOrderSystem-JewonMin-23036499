# 출고 처리 SPEC

## 개요

- 재고가 충분해진 CONFIRMED 주문에 대하여 출고를 처리
- 특정 주문에 대해 출고를 실행하면 주문 상태가 RELEASED로 전환

> 상태 전이 조건 전체는 [주문 상태 흐름](../order-status.md) 참고

---

## 처리 흐름

1. CONFIRMED 상태의 주문 목록 표시
2. 출고할 주문 번호 선택
3. 출고 처리 완료 → 주문 상태 CONFIRMED → RELEASED

---
