# 모니터링 SPEC

## 개요

- 주문에 따른 시료 재고에 대한 정보 모니터링 기능
- 담당자가 현재 시스템의 상태를 한눈에 파악할 수 있도록 구성

---

## 메뉴 구성

| 번호 | 메뉴명 | 설명 |
|------|--------|------|
| 1 | 주문량 확인 | 상태별 주문 건수 확인 |
| 2 | 재고량 확인 | 시료별 현재 재고 및 재고 상태 확인 |
| 0 | 위로 | 메인 메뉴로 복귀 |

---

## 주문량 확인

- 현재 상태별 주문 건수 표시
- 표시 대상 상태: RESERVED / CONFIRMED / PRODUCING / RELEASED
- REJECTED는 유효한 주문이 아니므로 제외

---

## 재고량 확인

각 시료별 현재 재고 수량 및 주문 대비 재고 상태 표기.

| 상태 | 조건 |
|------|------|
| 여유 | 주문 대비 재고 충분 |
| 부족 | 주문 대비 재고 수량 부족 |
| 고갈 | 재고 수량이 0 |

---
