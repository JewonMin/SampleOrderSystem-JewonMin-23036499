# 생산라인 SPEC

## 개요

- 생산라인에 대한 정보를 Display
- 주문량에 대한 부족분을 생산하되, 수율 및 오차를 고려하여 실 생산량을 산정
- 생산라인은 단일 라인 (1개), 스케줄링 전략: FIFO

---

## 실생산량 산정 공식

```
실생산량 = ceil(부족분 / (수율 × 0.9))
총 생산시간 = 평균 생산시간 × 실생산량
```

| 항목 | 설명 |
|------|------|
| 부족분 | 주문 수량 - 현재 재고 |
| 수율 | 정상 시료 수 / 총 생산 시료 수 (시료 속성값) |
| 0.9 | 오차 보정 매직 넘버. 수율만으로 계산한 생산량보다 여유분을 추가 확보하기 위한 안전 계수 |

> ex) 부족분 170 ea, 수율 0.92 → ceil(170 / (0.92 × 0.9)) = ceil(205.3) = **206 ea**  
> ex) 부족분 150 ea, 수율 0.88 → ceil(150 / (0.88 × 0.9)) = ceil(189.4) = **190 ea**

> 수율 정의는 [시료 관리](sample-management.md) 참고

---

## 생산 완료 처리

- 생산 완료 시 주문 상태: PRODUCING → CONFIRMED

---

## 화면 구성

### 생산 현황 표기

- 현재 생산 중인 시료에 대한 정보 표기
- 표기할 정보 수준은 자율적으로 결정
  - ex) 주문 정보, 현재까지의 생산량, 진행률, 완료 예정 시각 등

### 대기 주문 확인

- 생산 큐(FIFO)에 대기 중인 주문 목록 출력
- 표시 항목: 순서, 주문번호, 시료, 주문량, 부족분, 실생산량, 예상 완료 시각

---
