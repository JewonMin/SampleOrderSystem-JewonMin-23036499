# 시료 관리 SPEC

## 개요

- 시료(Sample)는 이 시스템의 가장 기본이 되는 단위
- 각 시료는 고유한 이름과 속성을 가지며, 시스템에 등록된 시료만 주문 가능

---

## 메뉴 구성

| 번호 | 메뉴명 | 설명 |
|------|--------|------|
| 1 | 시료 등록 | 새로운 시료를 시스템에 추가 |
| 2 | 시료 목록 | 등록된 모든 시료 목록 확인, 현재 재고 수량 함께 표시 |
| 3 | 시료 검색 | 이름 등 속성으로 특정 시료를 검색 |
| 0 | 위로 | 메인 메뉴로 복귀 |

---

## 시료 속성

| 속성 | 설명 |
|------|------|
| 시료 ID | 시료 고유 식별자 (예: S-001) |
| 이름 | 시료명 |
| 평균 생산시간 | 시료 1개 생산에 소요되는 평균 시간 (min/ea) |
| 수율 | 정상적인 시료 수 / 총 생산 시료 수 |

> * 수율 계산: 정상적인 시료 수 / 총 생산 시료 수  
> ex) 100개 생산 중 정상적인 물품 90개 = 0.9
