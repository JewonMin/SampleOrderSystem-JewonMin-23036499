# 시료 주문 SPEC

## 개요

- 고객이 시료를 요청하면 주문 담당자가 주문을 생성
- 주문 생성 시점의 상태는 RESERVED

---

## 시료 예약

- 고객이 원하는 시료와 수량을 주문
- 주문 접수 완료 시 고유 주문번호 발급

### 입력 값

| 항목 | 설명 |
|------|------|
| 시료 ID | 주문할 시료의 ID (등록된 시료만 가능) |
| 고객명 | 주문 고객명 |
| 주문 수량 | 요청 수량 (ea) |

### 주문번호 형식

```
ORD-YYYYMMDD-NNNN
예) ORD-20260416-0043
```

### 주문 생성 후 상태

| 항목 | 값 |
|------|----|
| 초기 상태 | RESERVED |
| 재고 확인 | [3] 주문 승인/거절 메뉴에서 진행 |

> 상태 흐름 전체는 [주문 상태 흐름](../order-status.md) 참고

---

## UI 화면 예시

```
============================================================
  [2] 시료 주문
------------------------------------------------------------
  시료 ID    > S-003
  고객명     > 삼성전자 파운드리
  주문 수량  > 200

  입력 내용 확인
  시료    SiC 파워기판-6인치  (S-003)
  고객    삼성전자 파운드리
  수량    200 ea

  [Y] 예약 접수    [N] 취소
  선택 > Y

  예약 접수 완료.

  주문번호    ORD-20260416-0043
  현재 상태   RESERVED

  ※ 재고 확인은 [3] 승인 메뉴에서 직접 진행하세요.

  선택 > _
```

> 위 화면 구성은 예시이며 자유롭게 표기 가능.
