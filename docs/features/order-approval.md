# 주문 승인/거절 SPEC

## 개요

- RESERVED 상태의 주문 목록을 확인하고 특정 주문에 대해 승인 또는 거절 처리
- 생산 담당자가 수행

---

## 접수된 주문 목록

- RESERVED 상태의 주문 목록만 표시

---

## 주문 승인

접수된 특정 주문에 대해 승인. 승인 시 재고 상황에 따라 2가지로 자동 처리.

| 재고 상황 | 처리 방식 | 전환 상태 |
|-----------|-----------|-----------|
| 재고 충분 | 주문을 즉시 처리 | RESERVED → CONFIRMED |
| 재고 부족 | 부족분을 생산 라인에 자동 등록 | RESERVED → PRODUCING |

### 재고 판단 기준

- 재고 충분/부족 판단은 **승인 처리 시점의 현재 재고** 기준
- 주문 접수(RESERVED) 시에는 재고를 선점하지 않음 → 동일 시료에 여러 주문이 있어도, 각 주문이 승인될 때마다 독립적으로 그 시점의 재고를 체크
- **현재 재고에는 생산라인에서 진행 중인 생산량이 실시간으로 포함됨** → 단일 생산라인에서 생산이 진행될수록 재고가 점진적으로 증가하며, 이후 승인되는 주문의 재고 판단에 반영됨

> 상태 전이 조건 전체는 [주문 상태 흐름](../order-status.md) 참고

---

## 주문 거절

- 접수된 특정 주문에 대해 거절
- 즉시 REJECTED 상태로 전환

---

```

  [3] 주문 승인/거절
------------------------------------------------------------
  승인 대기 중인 예약 목록  (RESERVED)

  번호    주문번호      고객               시료                    수량       상태
  -------------------------------------------------------------------------------
  [1]     ORD-0041      LG이노텍           산화막 웨이퍼-SiO2      300 ea     RESERVED
  [2]     ORD-0042      SK하이닉스         실리콘 웨이퍼-8인치     150 ea     RESERVED
  [3]     ORD-0043      삼성전자 파운드리  SiC 파워기판-6인치      200 ea     RESERVED

  승인할 번호 > 3

  재고 확인 중...

  시료        SiC 파워기판-6인치    현재 재고  30 ea
  주문 수량   200 ea                부족분     170 ea  ← 이 수량만 생산

  재고 부족.  부족분 170 ea 승인하시겠습니까?  (실생산량 206 ea / 165 min)

  [Y] 승인    [N] 주문 거절
  선택 > Y

  승인 완료.

  상태 변경   RESERVED  →  PRODUCING
  주문번호    ORD-20260416-0043
```

> 위 화면 구성은 예시이며 자유롭게 표기 가능.
